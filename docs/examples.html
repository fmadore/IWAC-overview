IWAC Database Overview - URL Examples

The IWAC Database Overview application supports URL parameters for direct access to specific language versions and visualization tabs. This page provides examples of how to use these parameters.

URL Parameters

The application supports the following URL parameters:

  • lang: Language code (en for English, fr for French)
  • tab: Visualization tab ID (countries, languages, timeline, types, categories, words)

Usage in Links

You can create links to specific views using the following format:

https://fmadore.github.io/IWAC-overview/index.html?lang=en&tab=types

This allows you to share direct links to specific visualizations in a specific language.
