This application includes enhanced debugging tools to help identify issues:

  • Filtering: Use the filter input to search for specific log entries
  • Download Logs: Click the download button to save logs for analysis
  • Browser Console Commands: The following commands are available in the browser console:
    • window.__IWAC_DEBUG.dumpLogs() - View all logs
